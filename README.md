# Endometriosis Risk Analysis Dashboard

Project Overview

This project presents a Power BI dashboard that analyzes diagnosis patterns of endometriosis using patient data such as BMI, age, chronic pain level, and symptom count.

The goal is to identify key risk factors and visualize trends for better understanding and decision-making.

---

 Objectives

* Analyze diagnosis trends across BMI categories
* Study the impact of chronic pain on diagnosis
* Explore patient demographics
* Build an interactive dashboard for insights

---

 Dashboard Features

* Heatmap of diagnosis rate (BMI vs Pain Level)
* Decomposition Tree for root cause analysis
* KPI Cards (Total Patients, Avg BMI, Diagnosis Rate, etc.)
* Interactive slicers (Age Group, BMI Category)
* Symptom-based diagnosis analysis

---

🔍 Key Insights

* Obese patients show the highest diagnosis rate (~51%)
* Diagnosis probability increases with chronic pain level
* Pain level is a stronger predictor than BMI
* High BMI + high pain = highest risk group
